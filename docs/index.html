An interactive fiction inspired by the original Half-Life, where you play as an engineer in an advanced government sponsored research facility.

Quantum Plex was created with Inform and has IFID FEDDB704-88F6-4758-AF7B-18F6881554F0. To play a work like this one, you need an interpreter program: many are available, among them Gargoyle for MacOS, Linux or Windows; Spatterlight for MacOS; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
